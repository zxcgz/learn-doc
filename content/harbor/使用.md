# 客户端

1.  注册帐号，进入Harbor登录页，注册帐号（需要管理员在后台开启允许注册）
    
2.  配置ca证书。将服务器端的ca证书下载到本地，并导入系统
    
3.  Windows
    
4.  Linux
    

```shell
# 
sudo cp /tmp/ca.crt /usr/local/share/ca-certificates/
sudo update-ca-certificates —fresh
# 麒麟系统
sudo cp /tmp/ca.crt /etc/pki/ca-trust/source/anchors/
# 或者
sudo cp /tmp/ca.crt /etc/pki/ca-trust/source/anchors/
sudo update-ca-trust
```

3.  重启Docker
    

```shell
sudo service docker restart
```

3.  登录Harbor仓库
    

```shell
docker login 192.168.7.114 -u username -p password
```

4.  拉取需要的镜像（在harbor没有配置上游仓库时，只能拉取目标harbor仓库中已有的镜像）
    

```shell
docker pull 192.168.7.114/zhang/cosign:v1.0
```

5.  上传镜像，上传镜像时需要将给镜像打上标签，包括域名（ip），镜像名，版本号等信息
    

```shell
#给镜像打标签
docker tag image 192.168.7.114/***/***:version
#推送
docker push 192.168.7.114/***/***:version
```
