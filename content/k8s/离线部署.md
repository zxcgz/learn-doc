# 离线部署

# kuboard

支持离线部署，但是不支持离线安装部署k8s集群，只能在已经安装有k8s集群的环境中离线安装

# kubeoperator

开源，支持离线部署k8s集群，离线包>6G，2024年7月之后已停止维护

# Sealos

支持离线部署k8s，使用类似Dockerfile的格式构建，目录格式类似Helm，是一个类似1Panel的管理集群的工具，将整个集群视为一个应用进行部署，

# KubeClipper

似乎可以离线部署k8s

# KubeKey

支持离线部署k8s，似乎可以将k8s打包成一个iso文件，也有提供[压缩包](https://juejin.cn/post/7350571075547791410)的方式

# kubespray

本身不支持离线部署k8s，可以通过搭建内网服务以及修改配置文件的方式实现离线部署k8s[博客](https://imroc.cc/kubernetes/basics/deploy/kubespray/offline)

# KuboardSpray

基于kubespray实现的可视化版本，提供离线包，支持离线部署k8s
